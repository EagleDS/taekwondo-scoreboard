Multi-Judge Setup

1
Create a free Firebase project
Go to console.firebase.google.com → Click Add project → Name it anything → Disable Google Analytics (optional) → Create project.
2
Add a Web App & copy config
In your project → Click the </> Web icon → Register app → Copy the firebaseConfig object values into the form below.
3
Enable Realtime Database
In the left menu → Build → Realtime Database → Create database → Choose any location → Start in test mode (you can secure it later).
4
Fill in the form & generate links
Enter your config below, give your session a name, hit Generate. Share the judge links to each judge's phone.
